# Proyecto personajes — Flexbox y diseño de personajes

Página de **personajes** con énfasis en **Flexbox** para alinear y ordenar tarjetas o bloques de contenido personalizados.

**Autora del repositorio:** [@Juli21v](https://github.com/Juli21v)

## Stack

- HTML5
- CSS3 (Flexbox en `CSS/style.css`)
- JavaScript (`JS/java.js`)

## Estructura del repositorio

```
Proyecto-personajes-/
├── index.html
├── CSS/style.css
├── JS/java.js
└── README.md
```

## Cómo verlo

Abre `index.html` en el navegador.

## Aprendizaje

Refuerzo de **layout moderno con CSS** sin depender solo de grids de frameworks.
